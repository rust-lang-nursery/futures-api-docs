pub struct Buffered<St: Stream> where
    St: Stream,
    St::Item: Future,  { /* fields omitted */ }

An adaptor for a stream of futures to execute the futures concurrently, if possible.

This adaptor will buffer up a list of pending futures, and then return their results in the order that they were pulled out of the original stream. This is created by the Stream::buffered method.
